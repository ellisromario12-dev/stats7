01 · concepts

The Big Three

Three ways to describe and summarize a data set — each tells you something different.

Md

Median

The middle value when all numbers are sorted in order. Half the values are above it, half are below.

Arrange → Find middle
  • 1Write the numbers in order from least to greatest
  • 2Find the middle value
  • 3If two middle values exist, average them
example
4, 7, 9, 11, 15 → Median = 9
Mo

Mode

The value that appears most frequently in the data set. A set can have no mode, one, or many.

Count repeats → Highest frequency
  • 1Count how many times each value appears
  • 2The value(s) with the highest count = mode
  • 3No repeats = no mode
example
3, 7, 2, 7, 9 → Mode = 7
R

Range

How spread out the data is — the difference between the largest and smallest values.

Range = Max − Min
  • 1Find the maximum (largest) value
  • 2Find the minimum (smallest) value
  • 3Subtract: Max − Min
example
3, 7, 2, 9, 14 → Range = 14 − 2 = 12

Even vs. Odd count

Odd number of values → one exact middle. Even number → average the n/2 and n/2 + 1 values.

06 · reference

Quick Reference

Median — key facts

  • Always sort first before finding the middle
  • Odd count: one middle value
  • Even count: average the two middle values
  • Outlier resistant — extreme values barely affect it
  • Good for skewed data like income or housing prices

Mode — key facts

  • No mode = all values appear exactly once
  • Bimodal = two modes (tie for highest frequency)
  • Multimodal = three or more modes
  • Only measure that works for non-numeric data
  • Useful for most popular item/category

Range — key facts

  • Always Range = Max − Min
  • Range can never be negative
  • Range of 0 means all values are equal
  • Highly sensitive to outliers
  • A larger range means more spread out data
